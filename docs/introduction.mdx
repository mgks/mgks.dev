# mgks.dev Documentation

Welcome to the official documentation for mgks.dev. This documentation provides comprehensive information about the website, its features, and how to contribute to the project.

## About mgks.dev

mgks.dev is a personal website and blog by Ghazi, an open source developer who builds simple solutions for complex technical problems. The website is built using Jekyll and hosted on GitHub Pages, with a focus on sharing knowledge about programming, open source projects, and technical problem-solving.

## Key Features

- **Personal Portfolio**: Showcasing projects, experience, and skills
- **Blog**: Articles about programming, open source, and technical solutions
- **Open Source**: The entire website is open source and available on GitHub
